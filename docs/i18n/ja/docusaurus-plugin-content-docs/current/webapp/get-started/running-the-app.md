# Running The App

フロントエンドをローカルで起動するには、次を実行します。

```bash
npm install
npm run dev
```

Vite の開発サーバーは、`/api` リクエストを `http://127.0.0.1:8001` の Julia バックエンドへプロキシします。

フロントエンドをビルドするには、次を実行します。

```bash
npm run build
```

これにより `dist/` が生成され、後で Julia バックエンドサービス向けにパッケージ化されます。

バックエンドが API キー保護されている場合、フロントエンドはキー入力を促し、その値をローカルストレージに保存します。
