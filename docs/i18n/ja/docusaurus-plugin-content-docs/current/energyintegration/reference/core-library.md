# Core Library

`EnergyIntegration` ライブラリは、この製品群の計算的な中心です。

この層はストリームデータや設定オブジェクトを次の形へ変換します。

- 構築可能な HEN 表現
- ソルバーに渡せる最適化問題
- バックエンドやフロントエンドで表示可能な結果オブジェクト

WebApp はこの層を置き換えるものではなく、Julia 側が生成する入出力を扱いやすくする UI です。

今後この共有ドキュメントサイトでは、このセクションが次の内容の置き場になります。

- Julia パッケージの導入
- コア API と型
- モデリング設定の詳細
- 求解とレポートの挙動
